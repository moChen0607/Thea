Compute the shortest paths in a graph from a source vertex (or a set of source vertices) to every other vertex, using Dijkstra's algorithm [1959].

A Fibonacci heap accelerates the computation to O(|E| + |V| log |V|) [Fredman & Tarjan 1984]. All edge lengths must be non-negative.

Parameters
graphThe graph to process. Must have non-negative edge lengths.
srcThe source vertex from which to measure distances. This vertex is initialized to distance zero and no predecessor. If src_region is non-empty, the src argument is ignored,
resultThe computed shortest-path information. Maps each vertex to the length of its shortest path from the source(s), as well its predecessor in the shortest path. Any prior data in the map is discarded.
limitIf set to a non-negative value, only returns vertices that are closer than this value.
src_regionIf non-empty, specifies a set of source vertices and the initial distances (must be non-negative) to them. In this case the src argument is ignored. The predecessor of each such vertex is absent, unless a shorter path to the vertex is found.
include_unreachableIf true, vertices unreachable from the source vertex are also returned, mapped to negative distances and without predecessors.

Compute the shortest paths in a graph from a source vertex (or a set of source vertices) to every other vertex, using Dijkstra's algorithm [1959].

A Fibonacci heap accelerates the computation to O(|E| + |V| log |V|) [Fredman & Tarjan 1984]. All edge lengths must be non-negative.

This version of the function calls a callback operation on each processed vertex once its distance from the source has been determined. The callback must be equivalent to the following function signature:

//
// vertex: The visited vertex in the graph.
// distance: Length of shortest path to the vertex from the source.
// has_pred: Does the vertex have a predecessor in the shortest path to the source?
// (False if the vertex is (part of) the source.)
// pred: Predecessor of the vertex in the shortest path, if has_pred is true, else an undefined value.
//
bool operator()(VertexHandle vertex, double distance, bool has_pred, VertexHandle pred);

The callback should normally return false, unless it wants to terminate the search, in which case it should return true.

Parameters
graphThe graph to process. Must have non-negative edge lengths.
srcThe source vertex from which to measure distances. This vertex is initialized to distance zero and no predecessor. If src_region is non-empty, the src argument is ignored,
callbackCalled for every visited vertex along with the distance to its shortest path from the source. If the function returns true, the search is terminated at this point. To pass a callback by reference, wrap it in std::ref.
limitIf set to a non-negative value, only returns vertices that are closer than this value.
src_regionIf non-empty, specifies a set of source vertices and the initial distances (must be non-negative) to them. In this case the src argument is ignored. The predecessor of each such vertex is absent, unless a shorter path to the vertex is found.
include_unreachableIf true, vertices unreachable from the source vertex are also returned, mapped to negative distances and without predecessors.

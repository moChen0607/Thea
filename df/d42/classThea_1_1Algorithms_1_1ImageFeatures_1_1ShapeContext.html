Detailed Description

Computes the shape context at pixels of a contour image.

If the image has more than one channel, the luminance channel is used. Follows the algorithm of:

Mori, Belongie and Malik, "Shape contexts enable efficient retrieval of similar shapes", CVPR 2001.

Computes the shape context at each pixel of a contour image.

Parameters
num_radial_binsNumber of divisions in the radial direction.
num_polar_binsNumber of divisions in the angular direction.
valuesComputed shape contexts, in a num_pixels * num_radial_bins * num_polar_bins array (num_pixels is major dimension for packing).
ignore_empty_pixelsDoes not compute shape contexts for pixels that are empty (zero luminance, i.e. black). The corresponding entries in values are set to zero.
max_radiusLimits the area of the context for a pixel to this radius (negative for default).

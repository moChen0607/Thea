Detailed Description

template<typename T, int N, typename ScalarT>
struct Thea::Algorithms::RayIntersectionTesterImpl< T, N, ScalarT >

Helper class for RayIntersectionTester.

Specializations of this class actually test for intersection. This is required because C++ does unexpected things with specialized and overloaded function templates (see http://www.gotw.ca/publications/mill17.htm).

The default implementation works for objects implementing the RayIntersectable3 interface.

Detailed Description

A set of options, specified as key-value pairs.

Supports a much more general set of value types than the simpler abstract interface it implements.

char const *const &  default_value

Specialization of get() for C-style strings.

In this particular case, a pointer to the C array underlying the held std::string is returned (which may be invalidated by further operations on this Options object). If the option has not been set, the default value specified by the last parameter is returned.

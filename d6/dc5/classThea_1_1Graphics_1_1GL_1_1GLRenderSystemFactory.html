RenderSystem * createRenderSystem ( char const *  name)
virtual

Create a rendersystem with the given name.

The rendersystem must be destroyed using destroyRenderSystem(). If no rendering context is available, the new rendersystem will be set up for offscreen rendering if possible. In this case, you must explictly create and attach a framebuffer to the rendersystem before you can call any drawing functions.

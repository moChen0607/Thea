Detailed Description

template<typename A, typename B, int N, typename T, typename Enable = void>
struct Thea::Algorithms::MetricL2Impl< A, B, N, T, Enable >

Helper class for MetricL2.

Specializations of this class actually compute the metric. This is required because C++ does unexpected things with specialized and overloaded function templates (see http://www.gotw.ca/publications/mill17.htm).

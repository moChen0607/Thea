Detailed Description

A reference to a vertex of a display mesh, via its index.

This is a convenience class that is better in some cases than DisplayMeshVertex (e.g. when references are created on the fly as vertices are added to the mesh (adding vertices invalidates previously created DisplayMeshVertex objects), or when memory conservation is important.).

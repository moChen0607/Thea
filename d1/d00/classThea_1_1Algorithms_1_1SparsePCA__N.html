Compute the sparse PCA axes of a set of N-dimensional objects.

InputIterator must dereference to type T or pointer-to-T.

Parameters
beginThe first object in the set.
endOne position beyond the last object in the set.
variancesThe variances of the data along the sparse PCA axes, sorted from maxium variances to least.
axesThe sparse PCA axes, sorted from maximum variance to least.
centroidIf non-null, used to return the centroid of the objects, which is computed during PCA.
lambdaThe bias towards sparsity. A negative value selects a default bias.
epsWhen the cost changes less than this between successive iterations, the search is assumed to have converged. A negative value selects a default threshold.
max_itersMaximum number of iterations. A negative value selects a default number.
Returns
True if the search converged, false otherwise.

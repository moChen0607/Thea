Detailed Description

Intersection queries on objects.

To add support for intersection queries between custom types, add specializations (full or partial) of the helper class IntersectionTesterImpl. This class is required because C++ does unexpected things with specialized and overloaded function templates (see http://www.gotw.ca/publications/mill17.htm). Note that to commutatively support queries between distinct types A and B, you must specialize both IntersectionTesterImpl<A, B> and IntersectionTesterImpl<B, A>. Do not try specializing IntersectionTester::intersects<N, T>().

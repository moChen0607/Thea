void computeWordFrequencies ( AddressableMatrixT const &  points,
U *  histogram,
double const *  point_weights = nullptr 
) const

Compute the histogram of word frequencies for a set of points.

Each point is assigned to its closest word, and the histogram bin corresponding to the word incremented by 1. Alternatively, a set of point weights may be specified, in which case the histogram bin is incremented by the weight of the point.

Parameters
pointsThe points for which to build the word histogram. AddressableMatrixT should have the interface of an AddressableMatrix of some real-valued scalar type.
histogramThe vector of word frequencies, assumed to be preallocated to numWords() entries. U should be a real-valued scalar type (e.g. intx or double).
point_weights[Optional] The contribution of each point to the histogram (1 if null).

bool train ( intx  num_words,
AddressableMatrixT const &  training_points 
)

Train the Bag-of-Words model from a set of training points, one per row of the input matrix.

The points are clustered into a vocabulary of num_words words during training.

Parameters
num_wordsNumber of words to cluster points into.
training_pointsThe set of training points, one vector per row. AddressableMatrixT should have the interface of an AddressableMatrix of some real-valued scalar type.
Returns
True if the training converged, else false.

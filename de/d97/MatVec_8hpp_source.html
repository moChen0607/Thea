32 // Typedef common instantiations. These are fully instantiated and have no further template parameters.
33 // Alignment is DISABLED for fixed-size vectorizable types (Vector4f, Matrix4f, Vector2d etc). These require special handling
34 // for every class which directly or indirectly has such a member, and could lead to unexpected bugs and considerably less ease
35 // of use.
36 //
37 // See:
38 // - http://eigen.tuxfamily.org/dox/group__TopicStructHavingEigenMembers.html
39 // - https://eigen.tuxfamily.org/dox/group__TopicStlContainers.html
40 // - http://eigen.tuxfamily.org/dox/group__TopicFixedSizeVectorizable.html
41 // - https://stackoverflow.com/q/41087043
42 //

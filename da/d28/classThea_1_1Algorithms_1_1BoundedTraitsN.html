Detailed Description

template<typename T, int N, typename ScalarT = Real, typename Enable = void>
class Thea::Algorithms::BoundedTraitsN< T, N, ScalarT, Enable >

Traits class for a bounded object in N-space.

Default implementation assumes class T has a T::getBounds() function that returns an N-dimensional axis-aligned bounding box. Different specializations exist for specific classes.

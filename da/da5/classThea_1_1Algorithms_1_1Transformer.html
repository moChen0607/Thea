Detailed Description

Apply transformations to objects.

To add support for custom object and transformation types, add specializations (full or partial) of the helper class TransformerImpl. This class is required because C++ does unexpected things with specialized and overloaded function templates (see http://www.gotw.ca/publications/mill17.htm). Do not try specializing Transformer::transform().

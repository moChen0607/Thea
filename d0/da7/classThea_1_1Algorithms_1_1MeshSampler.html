Samples approximately uniformly separately points from the mesh.

The function returns an array of samples ordered so that for any K, the first K samples form an approximately uniformly separated subsampling. The function initially computes an oversampling of the mesh by area.

Parameters
desired_num_samplesThe number of samples desired. This need not be exactly the number actually computed (and returned by the function), unless count_mode == CountMode::EXACT.
positionsUsed to return the sample positions.
face_normalsIf not null, used to return the normals of the parent faces of the samples.
trianglesIf not null, used to return the mesh triangles from which the samples were selected.
count_modeIf CountMode::EXACT, exactly desired_num_samples samples are returned. Else, approximately desired_num_samples samples are returned. The latter is likely to be faster.
oversampling_factorThe ratio of the number of initially selected points to the number of finally returned points. The default, selected with a negative value, is 3.
verboseIf true, prints progress messages.
Returns
The number of samples computed.

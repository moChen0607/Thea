Detailed Description

template<typename MeshT>
class Thea::Algorithms::MeshTriangles< MeshT >

A set of triangles obtained by triangulating mesh faces.

Implemented for general, DCEL and display meshes.

See also
GeneralMesh, DCELMesh, DisplayMesh

Detailed Description

template<typename ObjectT, typename TransformT>
class Thea::Algorithms::TransformedObject< ObjectT, TransformT >

Wraps pointers to an object and an associated transformation.

The pointer values will be dereferenced as necessary by the get...() functions. The original data must persist until this class is destroyed. One should always check the return value of the has...() function before calling the corresponding get...() functions.

Detailed Description

A boosting classifier using shared features.

Based on

A. Torralba, K. Murphy and W. Freeman, "Sharing features: efficient boosting procedures for multiclass object detection", Proc. CVPR, 2004.

To use the class, implement an appropriate subclass of TrainingData, call train(), and then call predict().

Detailed Description

Proximity graph on surface samples.

Satisfies the IsAdjacencyGraph concept. Typical usage is to first specify the sample positions (and optionally normals) via setSamples(). Then, specify an optional dense oversampling via setOversampling() – the oversampling makes it easier to verify if two samples are actually adjacent on the surface. Finally, call init(), optionally with a representation of the underlying surface, to compute the sample adjacencies (graph edges).

Note
The graph is not created until init() is called.

Real getAverageSeparation ( ) const

Get the average separation of samples from their neighbors, that is, the average edge length of the graph.

This may be computed by sampling and hence should not be assumed to be a true average, just a representative value. Has a meaningful value only after graph initialization, i.e. calling init(). It should not be called before then.

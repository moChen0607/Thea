Detailed Description

Mesh an implicit surface defined as the zero level set of a 3D function.

The function type F must be

<real-number-type> (*F)(Vector3 const & p)

or (if it is a functor class) must have the public member function

<real-number-type> operator()(Vector3 const & p) const

Polygonize the zero level set of a 3D function to a mesh, using the method of Bloomenthal [1994].

Jules Bloomenthal, "An implicit surface polygonizer", Graphics Gems IV (P. Heckbert, ed.), Academic Press, New York, 1994.

Parameters
surface_functorThe function whose zero level set defines the surface.
bounding_ballBounds the surface.
pt_near_surfaceA point on or near the zero level set.
optionsOptions controlling mesh generation.
resultThe output mesh will be stored here (any prior data will not be removed from the mesh).

Polygonize the zero level set of a 3D function to a mesh, using the method of Boissonnat and Oudot [2005] as implemented by the CGAL library (http://www.cgal.org/Manual/latest/doc_html/cgal_manual/Surface_mesher/Chapter_main.html)

Jean-Daniel Boissonnat and Steve Oudot, "Provably good sampling and meshing of surfaces", Graphical Models, 67:405-451, 2005.

Parameters
surface_functorThe function whose zero level set defines the surface.
bounding_ballBounds the surface. The functor must evaluate to a negative value at its center.
optionsOptions controlling mesh generation.
resultThe output mesh will be stored here (any prior data will not be removed from the mesh).

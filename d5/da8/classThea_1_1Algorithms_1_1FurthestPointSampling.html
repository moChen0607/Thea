Detailed Description

Subsample a set of 3D points to pick a subset of points distributed evenly, that is, neighboring points are (roughly) equally spaced.

The algorithm repeatedly picks the left-over point that is furthest from any of the previously selected ones. The returned list of points has the property that any prefix of the list is also an evenly spaced set, making further subsampling trivial.

Samples approximately uniformly separated points from a larger set.

The algorithm repeatedly picks the left-over point that is furthest from any of the previously selected ones. The function returns an array of points ordered so that for any K, the first K points form an approximately uniformly separated subsampling.

Parameters
num_orig_pointsThe number of input points.
orig_pointsThe set of input points.
num_desired_pointsThe number of points to be subsampled.
selected_indicesThe indices of the subsampled points. This array must be preallocated to (at least) num_desired_points elements.
dist_typeThe distance metric to be used. Currently only DistanceType::GEODESIC is supported.
verboseIf true, prints progress messages.
Returns
The number of subsampled points. A negative value, or a value less than num_desired_points in general, indicates an error occurred.

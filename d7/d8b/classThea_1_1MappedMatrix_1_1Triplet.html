Detailed Description

template<typename T = Real, typename IndexT = intx>
class Thea::MappedMatrix< T, IndexT >::Triplet

The stored (row, col, value) triplet type.

We use a set of triplets, rather than a (row, col) –> value map, for easy interoperability with the SparseMatrix::setFromTriplets() function. Implementionally, the two are similar since std::unordered_map is a set of (key, value) pairs.

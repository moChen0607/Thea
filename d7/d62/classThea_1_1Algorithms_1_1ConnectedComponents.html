Compute all maximal edge-connected components of a general mesh.

Two faces are edge-connected if they share an edge. An edge-connected path is a sequence of faces such that every consecutive pair is edge-connected. An edge-connected component is a set of faces such that every pair is connected by an edge-connected path consisting only of faces from the set.

Returns
The number of components found (== components.size())

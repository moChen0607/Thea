Detailed Description

template<int N, typename T = Real>
class Thea::Algorithms::ProximityQueryStructureN< N, T >::NeighborPair

A return value of a k-nearest-neighbors query, specified by a monotone approximation to (for L2, square of) the distance between neighboring elements, the indices of the query and target elements, and the positions of the closest pair of points on them.

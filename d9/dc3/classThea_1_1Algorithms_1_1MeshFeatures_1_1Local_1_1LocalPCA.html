Detailed Description

template<typename ExternalSampleKDTreeT = KDTreeN<Vector3, 3>>
class Thea::Algorithms::MeshFeatures::Local::LocalPCA< ExternalSampleKDTreeT >

Compute local PCA features at a point on a mesh.

The local PCA features are eigenvalues of the distribution of samples in the neighborhood of the point, sorted in decreasing order. Optionally, the eigenvectors (principal components) of the distribution may also be returned.

Compute the PCA features at a query point on the mesh.

Parameters
positionPoint at which to compute features.
eigenvectorsIf non-null, used to return eigenvectors of samples in the neighborhood, sorted in order of decreasing eigenvalue. Must be pre-allocated to (at least) 3 elements.
nbd_radiusThe size of the local neighborhood for which the features are computed, specified as a multiple of the shape scale. A negative argument selects a default size.
Returns
Eigenvalues of samples in the neighborhood, sorted in decreasing order.

Detailed Description

template<typename T, int N, typename Enable = void>
class Thea::Algorithms::IsBoundedN< T, N, Enable >

Has boolean member value = true if T is a geometrically bounded object in n-D space, else value = false.

Unless you specialize the class to set the value to true, it is false by default.
